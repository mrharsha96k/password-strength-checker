#  Password Strength Checker

A Python-based command-line tool to analyze password strength and provide suggestions for creating stronger passwords.

##  Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects digits and special characters
- Rates strength as Weak, Medium, or Strong
- Provides suggestions to improve weak passwords

##  Tech Stack
- Python 3
- re (regular expressions)

##  How to Run
```bash
python checker.py
